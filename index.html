Jig: An Animation Library

Jig is an instantiated object containg an individual timeline of TimelineLite. The result is the ability to write preset animation sequences as functions. Jig is an idea manifested, it is meant to speed up the production of nifty little animations for websites or applications. Also as demonstrated by 'WIGGLE' you can set your animations in beats per minute, potentially syncronizing them with music!

wiggle( element, {parameters});

wiggle('wiggle',{bpm:120, ease:'easeInOut', rotation:20, aloofness:20, amplitude:20, repeat:'forever'});
jump( element, {parameters});

jump('jump',{aloofness:20,exaggeration:10,amplitude:140, speed:.4, repeat:'forever'});
spin, element, {parameters});

spin('spin',{rotation:360, speed:2, repeat:'forever'});
pulse(element, {paramters})

pulse('pulse',{endScale:.8, speed:.5, repeat:'forever'})
fly.from( element, {parameters});

fly.from('fly0',{startX:50, startY:-50, startScale:0, rotation:40, speed:.5, ease:'easeOutExpo',startOpacity:0});
plop( element, {parameters});

each(['plop1','plop2','plop3'],plop, .2, {speed:.6,startY:'800px',endY:0, exaggeration:10});
