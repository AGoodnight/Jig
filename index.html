Delay

You can enter a delay, this will be how long the animation waits before starting relative to the last animation you assigned to the timeline through chaining.

jig('.className').hop(1)

Zigs in Jigs

Since a zig is an object you can treat it as such in your browser when debugging or animatig complex situations.

var myJig = jig('.className').hop().plop().fade();
console.log(myJig.zigs);




Preset Zigs

Preset Zigs are stored in the library.js file. They are built to stand-alone without any user input. A user can include user defined options though. These preset methods are speciffcations of the more general class of zig.



Options

Options are passed to the preset method in the form of an object literal {}. A user cannot introduce functionality that isnot there, but they can overide the variables controling the preset animation. Each preset acccepts it's own set of additional options alongside the base options of preset zig class.


Base Options Type Description
name 'string' If you prefer to be subjective
speed 'number' How fast the full animation is in seconds
stagger 'number' The delay of each successive element in the jig
repeat 'number' forever is 99999999999999999
